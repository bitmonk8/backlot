## Project-Specific Labels

Assign exactly one crate label per issue based on file path prefix.

| Label | Path prefix |
|-------|-------------|
| `crate:flick` | `flick/` |
| `crate:lot` | `lot/` |
| `crate:reel` | `reel/` |
| `crate:vault` | `vault/` |
| `crate:epic` | `epic/` |
| `crate:mech` | `mech/` |

If the issue spans multiple crates, use the crate of the primary file. If no crate matches, omit the crate label.
